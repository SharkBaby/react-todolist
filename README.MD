# react-todolist

react-todolist is a react sample for react learners. This demo shows the basic technology of building a simple react project.

+ redux
+ webpack
+ async/task
+ others

## How to use
+ `npm run build`
+ `npm start`
+ open browser and visit `localhost:8080`
